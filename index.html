Minhas Especialidades

Back-end

Python

Proficiente em Python, aplicando a linguagem em desenvolvimento back-end e automação de tarefas. Tenho experiência com APIs, integração de bancos de dados e processamento assíncrono, sempre buscando eficiência e código otimizado.

Django

Experiência com Django no desenvolvimento de aplicações web completas. Domínio de Django ORM, autenticação, autorização e APIs RESTful, criando soluções escaláveis e seguras.

PostgreSQL

Experiência com modelagem de dados, relacionamentos, consultas SQL otimizadas e uso de funções avançadas no PostgreSQL. Foco em performance e integridade dos dados.

Docker

Conhecimentos em containerização de aplicações com Docker, criação de imagens, gerenciamento de containers e uso de Docker Compose para ambientes de desenvolvimento integrados.

Front-end

JavaScript

Experiência sólida em JavaScript para criar interações dinâmicas e funcionalidades avançadas. Domínio em manipulação do DOM, eventos e integração com APIs externas.

React

Proficiência em React para desenvolvimento de interfaces modernas e reativas. Experiência com hooks, gerenciamento de estado (Context API), roteamento e componentização reutilizável.

Experiência com estilização avançada utilizando SASS, componentes visuais com Bootstrap e automação de tarefas com Gulp, visando produtividade, organização e responsividade no front-end.

HTML & CSS
